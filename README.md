# AI Quiz Generator

A terminal‑based Python quiz application that asks questions, accepts answers, validates them, calculates scores, and saves results.

## Features (All mandatory + optional)

- ✅ Welcome message
- ✅ Minimum 5 quiz questions (easily changeable)
- ✅ User answer input (with MCQ letter support)
- ✅ Answer validation
- ✅ Score calculation
- ✅ Loop for multiple questions
- ✅ Final result display with grade
- ✅ Score saved to `scores.txt`
- ✅ Timer (10 seconds per question)
- ✅ Multiple choice questions
- ✅ Random question order
- ✅ Score history viewer

## How to Run

1. Save all four files (`quiz.py`, `questions.txt`, `scores.txt`, `quiz_optional-features .py`,`README.md`) in one folder.
2. Open a terminal / command prompt **in that folder**.

   python quiz.py
